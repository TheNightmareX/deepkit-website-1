Isomorphic TypeScript

We believe in TypeScript. So much that we truely think you can build almost anything with just one tech stack and use native addons or WebAssembly where necessary. Isomorphic TypeScript allows you to write frontend, backend, cli tools, micro-service, apps, and much more with the very same language. Isomorphic TypeScript’s potential is huge and has not yet been utilised completely by other frameworks.

We want to change that and provide solutions, libraries, and tools to allow you to maximal utilise the full potential of TypeScript in your applications. Correctly applied you are an order of magnite faster compared to classic dual stacks like PHP/JS, or Java/JS.

Frontend agnostic

Deepkit is fully agnostic to any frontend framework so you can progressively adopt Deepkit libraries in your already existing or new Angular, React, VueJs application or jump directly to the full Deepkit framework in combination with a frontend framework of your choice.

That being said, Deepkit comes also with an optional UI framework based on Angular for desktop applications.

High performance

We love high performance code. Google’s v8 JavaScript runtime is very performant if the right code is provided, thats why we try to write always the most performant code possible. We have performance always in our top prioritities.

Deepkit Type is the fastest TypeScript serializer and very fast validator, Deepkit ORM the fastest TypeScript ORM for SQL databases and MongoDB, and Deepkit BSON the fastest BSON serializer. We are committed to provide even more top-notch high performance libraries in the future.

Real-time

More and more applications provide real-time functionality - and users expect it more and more. Deepkit comes in its DNA equipped with real-time functionality. Deepkit RPC for example does not only provide native support for RxJS, but also real-time collections and entities. Deepkit ORM has a LiveDatabase abstraction that allows you to stream changes made to your database directly to your subscribers (for example frontend).
