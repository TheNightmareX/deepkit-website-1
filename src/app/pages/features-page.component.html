Isomorphic TypeScript

We believe you can build almost anything with just one tech stack and use native addons or WebAssembly where necessary. Isomorphic TypeScript allows you to write frontend, backend, cli tools, micro-service, apps, and much more with the very same language. Its potential is huge and has not yet been utilised by the industry.

We want to change that and provide solutions, libraries, and tools that are highly optimised to interoperate perfectly together for frontend as well as backend applications.

Frontend agnostic

Deepkit is fully agnostic to any frontend framework so you can progressively adopt Deepkit libraries in your already existing projects or jump directly to the full Deepkit framework in combination with a frontend framework of your choice.

High performance

We love high performance. Not only in regards to execution time, but also development speed, and time to market. We are committed to build a framework that utilises the power of isomorphic TypeScript and a ecosystem of libraries that are perfectly optimised to work with each other to make you build applications faster than ever before.

Real-time

More and more applications provide real-time functionality - and users expect it more and more. Deepkit comes in its DNA equipped with real-time functionality. Deepkit RPC for example does not only provide native support for RxJS, but also real-time collections and entities. Deepkit ORM has a LiveDatabase abstraction that allows you to stream changes made to your database directly to your subscribers (for example frontend).
